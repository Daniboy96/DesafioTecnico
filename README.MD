# Automação Site Zitrus

## Descrição
Repositório de automação em Cypress para teste do desafio técnico.


## Como instalar

Garanta que esteja usando Node v16 (se preferivel usar nvm para gerenciar versões)
```
node --version
```

Instalar
```
npm install
```

## Como rodar

**UI** 
```
npx cypress open
```
